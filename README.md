 My personal configuration for river wm

- river (with Xwayland support)
- waybar
- mako
- swaylock
- swayidle

- i3-battery-popup (optional)
- grim
- slurp
- wl-clipboard
- wofi (or others)
- brightnessctl (only laptop)
- pamixer
